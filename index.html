Python is an interpreted, object-oriented, high-level programming language with dynamic semantics.

Python dibuat oleh Guido Van Rossum, edisi pertama python dirilis pada tahun 1990 (v.0.1.0)

Konversi Tipe Data

Meskipun python telah otomatis mendeteksi jenis tipe data pada variabel, ada kalanya kita perlu merubah tipe data tersebut.

Contoh: 

>>> a = 19
>>> b = 5
>>> a / b
3 # hasil akan 3 karena tipe data dari variable a dan b adalah integer


>>> a = 19
>>> b = 5
>>> float(a) / float(b)
3.8 # hasil 3.8 karena tipe data adalah float (decimal)
Fungsi float() akan merubah nilai a menjadi 19.0 dan b menjadi 5.0

Fungsi-fungsi untuk mengubah tipe data:

  • int() untuk mengubah menjadi integer;
  • long() untuk mengubah menjadi integer panjang;
  • float() untuk mengubah menjadi float;
  • bool() untuk mengubah menjadi boolean;
  • chr() untuk mengubah menjadi karakter;
  • str() untuk mengubah menjadi string.
  • bin() untuk mengubah menjadi bilangan Biner.
  • hex() untuk mengubah menjadi bilangan Heksadesimal.
  • oct() untuk mengubah menjadi bilangan okta.

*args dan **kwargs pada dasarnya adalah sebuah variabel. Tetapi terkadang juga disebut dengan magic variable, karena kemampuannya yang aneh.

Tetapi *args dan **kwargs digunakan sebagai parameter pada function.

Contoh: 

def call_name(*name):
  print "List name"
  for n in name:
    print n

call_name('john', 'doe')
Note: *args dan **kwargs adalah bukan nama yang baku. kita bisa menggunakan nama lain seperti *name, **name, *bio, etc

Perbedaan dari *args dan **kwargs

Perbedaan yang mudah di lihat adalah *args menggunakan satu tanda bintang, sedangkan **kwargs menggunakan dua tanda bintang.

  • * bintang satu artinya akan membuat sebuah parameter dengan tipe tuple
  • ** bintang dua artinya akan membuat sebuah parameter dengan tipe dictionary

# buat *args
def call_name(*name):
  print name

# buat **kwargs
def biodata(**bio):
  print bio

# call function
call_name('john', 'doe')
biodata(name='edi', address='jakarta')

# output
('john', 'doe')
{'name': 'edi', 'address': 'jakarta'}

Terdapat beberapa jenis mode dalam pengelohan file, berikut daftar mode tersebut

  • r: read atau hanya baca saja
  • w: write atau untuk menulis file, jika data file sudah ada akan di replace

  • r+: read + atau digunakan untuk membaca sekaligus untuk menulis data ke dalam file
  • b: binary atau sering digunakan untuk binary file terutama pada windows

class PersegiPanjang:
  ''' Class Persegi Panjang

  Ini docstring
  '''
  
  def __init__(self, panjang, lebar):
    self.panjang = panjang
    self.lebar = lebar

Python Inheritance


class Mencetak:
  
  def set_kata(self, kata):
    self.kata = kata

  def cetak(self):
    print "Kata yang ingin anda cetak :", self.kata


class Printer(Mencetak):

  def tambah_kata(self, kata):
    self.kata = self.kata + kata


c = Mencetak()
c.set_kata('hay')
c.cetak()

p = Printer()
p.set_kata("Buku Python")
p.tambah_kata(" Bisa beli di amazon.com")
p.cetak()
another example: 

class Parent(object):

  def override(self):
    print "PARENT override()"

  def implicit(self):
    print "PARENT implicit()"

  def altered(self):
    print "PARENT altered()"


class Child(Parent):

  def override(self):
    print "CHILD override()"

  def altered(self):
    print "CHILD, BEFORE PARENT altered()"
    super(Child, self).altered()
    print "CHILD, AFTER PARENT altered()"


dad = Parent()
son = Child()

dad.implicit()
son.implicit()

dad.override()
son.override()

dad.altered()
son.altered()

Note: super() digunakan untuk memanggil ulang method dari parent class. sehingga kita bisa memodifikasi method pada class children

Testing pada python memeliki beberapa library, pada contoh ini hanya menggunakan unittest test bawaan python.

test_example.py 

import unittest

class TestStringMethods(unittest.TestCase):

    def test_upper(self):
        self.assertEqual('foo'.upper(), 'FOO')

    def test_isupper(self):
        self.assertTrue('FOO'.isupper())
        self.assertFalse('Foo'.isupper())

    def test_split(self):
        s = 'hello world'
        self.assertEqual(s.split(), ['hello', 'world'])
        # check that s.split fails when the separator is not a string
        with self.assertRaises(TypeError):
            s.split(2)

if __name__ == '__main__':
    unittest.main()

2. Virtualenv

virtualenv is a tool to create isolated Python environments.

Jadi mudahnya virtualenv ini digunakan untuk membuat environment python yang terisolasi yang berfungsi untuk memisahkan module-module / package dari python.

Instalation 
sudo pip install virtualenv
Contoh penggunaan: 
# create folder
$ mkdir my-app
$ cd my-app

# create environment
$ virtualenv env

# activate environment
$ source env/bin/activate

# install a package
(env)$ pip install flask

# deactivate environment
$ deactivate
